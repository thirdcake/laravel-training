## 概要

Laravel の機能を学習するサイトです。

すでにさまざまな学習サイトがありますが、当サイトの特徴は、演習形式であることと、テスト駆動であることです。  
問題が与えられ、自分でテストを書き、失敗させ、成功するように変更し、リファクタリングします。

## フィードバックについて

当サイトの管理人もまた、 Laravel の入門者です。  
掲載内容には細心の注意を払っておりますが、間違いや勘違いが含まれている可能性があります。

もし不備を見つけた際には、 GitHub issues や、お問い合わせフォームからお知らせいただけると幸いです。

## 免責事項

当サイトの管理人は Laravel の入門者ですので、内容が正確でない場合があります。  
また、環境の違いにより、管理人が試した内容と、利用者のローカル環境で、異なる結果になることもあります。  
当サイトの内容を試して、何らかの不具合があったとしても、一切の責任は負えません。  
特に、 Laravel では、コマンドの実行を多用しますが、ローカル環境での実行の際には、よく内容を吟味し、自己責任で実行してください。

## 著作権

当サイトに掲載された内容の著作権は、当サイトの管理人に帰属します。  
問題文などを引用される際には、当該ページへリンクを貼っていただけると大変うれしいです。

## 禁止事項

当サイトは、比較的安いレンタルサーバーで運用しています。  
サーバーへの高い負荷のかかる行為は禁止します。  
特に、スクレイピングなどの実行は、やめてください。

当サイトの内容は、 GitHub のパブリックリポジトリにあります。  
Hugo という SSG で生成していますので、 `content` というフォルダ内にすべての情報のマークダウンが入っています。
もし内容を検索したい場合は、 git clone して、ローカルで find / grep コマンドなどで探してください。

## プライバシーについて

当サイトでは、アクセス時にアクセスログを取得しています。  
アクセスログには、 IP アドレスなどが含まれます。

当サイトのお問い合わせページから取得された情報については、お問い合わせ対応及び、サイトの改善を目的として利用します。  
また、個人情報等については、関係法令を遵守し、適切に管理します。  
これらの個人情報等は、関係法令による定めや所管官庁の命令に基づき、開示されることがあります。
